Data-attribute

It's use for embedding data in a web page, usign attributes that are ignored by the browser

Their purpose is to be used by JavaScript

The names of these attributes can be decided by the programmer, but must follow this format:

  • Start with data-
  • Only lowercase letter, number, dots, colons or underscore
  • Include an optional string value.

Example: 

                    data-power = 'flight superspeed'
                    data-user = 'DIEGO'

This informations on the attributes can be used to identify particular elements.

Each element has a dataset property to access the data-

                <div id='hero' data-powers='flight superSpeed'>
                    Superman
                </div>


                const superman = document.getElementById('hero');
                const powers = superman.dataset.powers;
                Output= 'flight superSpeed'

Web Storage

The W.S. provides key-value store on the client's computer that is similar to using cookies but has fewer restrictions, more storage capacity, and is generally easier to use.

The information store is not shared with the server on every request

The information is available on the windows who's domain is the same.

Storage capacity have a limit set al 5GB per domain

Example: 

                localStorage.setItem('name', 'Walter White');
                
                To Get the data back:
                localStorage.getItem('name'); 
                Output= "Walter White"

Also: 

                localStorage.name = 'Heisenberg'; 

                console.log(localStorage.name); 
                Output "Heisenberg";

To Remove items: 

                localStorage.removeItem('name');   
                OR
                delete localStorage.name;
                
                To Complete remove everything:
                localStorage.clear();

We can use a JSONs files: 

                localStorage.setItem('superman', JSON.stringify(hero);  
                and
                superman = JSON.parse(localStorage.getItem('superman'));

GeoLocation

Is used to obtain the geographical position of the device. User exact location.

You can link the user to nearby places, measure the speed which the user is moving.

Its a property of the navigator object, and it has a method called getCurrentPosition()

                navigator.geolocation.getCurrentPosition(youAreHere);

Thet will return a position object called youAreHere. The position object has a coords property with a Latitud and Longitud property

                function youAreHere(position) {
                    console.log(`Latitude: ${position.coords.latitude}, Longitude: ${position.coords.longitude}`); 
                 }

It has other properties: position.speed, position.altitude, position.heading and position.timestamp.

Web Workers

JavaScript is sigle-=threaded lenguage, Web workers allow processes to be run in the background, adding support for concurrency.

We use the worker() contructor: 

                const worker = new Worker('task.js');

The task.js file will be downloaded asynchronously.

The postMessage() method can be used to send a message and start the worker working

When the worker complete its task, it can be stopped using terminate() method or using close()
