About the WebSite

We created a website to adopt dogs. The page is simple and easy to navigate. When accessing the page, we will find the home page in which it will show us the photo of 4 dogs which we can choose to adopt. After we choose one, it will take us to a second page in which it will show us a card with data about the dog we chose. If we finally decide to adopt it, it will take us to a form that we must fill out to adopt the dog.

In the navigation menu we have other options such as food and donate. Food will take us to a page where we can put an ingredient and it will give us food recipes. Donate will take us to a page that is a form where we can fill out to donate a dog.

Description of the app

We create JSON that contains the data of the dogs to be adopted. When we enter the home page it will take the data and put it in the photos. For that we create a file called data-source that will be in charge of taking the JSON and copying it in the LocalSotorage. Then we take the data from localStorage and enter it in each of the photos of the dogs that will be adopted.

It will then save the user's selection and take that value from localStorage to place on the cards. The information that users enter in the form will also be saved in the LocalStorage.

We use API so that when someone searches for an ingredient, they can find food recipes with that ingredient. We use API from Edaman.

The page style has shadows, transition, hover, transform use of flex and grid, before and animations.
