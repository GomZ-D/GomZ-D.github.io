Week 02 - Lesson 03

The Window Object

Collection of properties and methods that contains information about the browser and computer. There are available through the Windows Object. Global variables are actual properties of a Global Object (windows Object)

Dialogs

Not part of EcmaScript, but of windows object: Alert(), Confirm() and prompt().

Browser Information

Navigation properties that reference the navigator object: Information of the browser, user Agent properties and information of the operation system.

Location:

Windows.location, location.href, location.protocol, location.port, location.host, location.hostname, location.pathname, location.search, location.hash, location.reload(), location.assign(), location.replace(), location.toString().

Browser History

Access to the browser history:

Window.history.length: How many page have you visited.

window.history.go(): go to a specific page.

window.history.forward() and history.back()

Controlling Windows

To control the windows of the web browser.

Window.open(): opens a page. Windows.close(): Close a windows if you have a reference of it.

Window.moveTo(0,0): to move a window. Window.resizeTo(0,0): Width, height of a widows.

Screen Information

This are more useful on mobile to detect a change in its orientation. Window.screen: give us the information about the screen of the browser. Window.screen.heigh, window.screen.width, window.screen.availWidth, window.screen.availHeight, window.screen.colorDepth.

Document Object

It has the properties or methods that deal with the page that has been loaded into the window.

Document.write() : write a string of text

Cookies

Small files saved locally on a user’s computer. Store information that can be retrieve between requests. Just can be read by a webpage of the same domain. 4kb of data. It contains list of “name:value”.

Document.cookie= ‘name:diego’;

Templates

Mechanism for holding HTML that is not to be rendered immediately when a page is loaded, but rendered after with JS.

For subsequent use in the document.Template element supports global attributes.

Avoiding DocumentFragment

Always preferable to clone or refer to the elements within it.

Example:

<div id="container"></div>
<template id="template">
<div>Click me</div>
</template> "
