Transitions

Transitions allow the values of CSS properties to change over time

Steps to create transition:

  • Declare the original state of the element in the default Style declaration
  • Declare the final state of your transitioned element(for example: hover)
  • Include the transition function in your default style.

Animations

A keyframe is a snapshot that defines a starting or end point of any smooth transition CSS animations allow us to add any number of keyframes in between, to guide our animation in more complex ways

To animate an element you need first to create a named animation, then attach it to an element. To create the animation use the @keyframes. Use WebKit follow by a name of your choosing.

Example: 

                @-webkit-keyframes myAnimation { 
                    /* put animation keyframes here */
                }
                @keyframes myAnimation { 
                    /* put animation keyframes here */
                }
